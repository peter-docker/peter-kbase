Access to the [Docker support site](https://support.docker.com "https://support.docker.com") is reserved for our customers with a subscription to Docker EE, Docker Enterprise Edition Engine, Universal Control Plane, or Docker Datacenter.

To gain access to the [Docker support site](https://support.docker.com "https://support.docker.com"), a Support Administrator must open a support ticket at [s](https://support.docker.com "https://support.docker.com")[upport.docker.com](https://support.docker.com/ "https://support.docker.com/") to request to add a support account member.

Please include the name, Docker ID, email address, and email address.

Once the request has been received, a support account will be created.

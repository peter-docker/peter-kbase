Docker Datacenter is now part of the Docker EE Standard offering. Docker EE is available in three tiers: **Basic** comes with the Docker platform, support and certification, and **Standard** and **Advanced** tiers add advanced container management (Docker Datacenter) and Docker Security Scanning.

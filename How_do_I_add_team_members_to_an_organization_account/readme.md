If you have an organization account and you would like to add a team member follow these steps.

## Docker Hub

![Edit section](./images/icon-trans.gif)

For Docker Hub follow these steps

1. Log in to your[ Docker Hub](https://hub.docker.com/ "https://hub.docker.com") account.
2. Click **Organizations** in the top navigation bar.
3. Select the organization that you would like to add a member too.
4. Select **Teams** in the secondary navigation bar.
5. Type in the Docker ID of the account you would like to add in the grey text bar.

## Docker Cloud

![Edit section](./images/icon-trans.gif)

For Docker Cloud follow these steps:

1. Log in to your [Docker Cloud](https://cloud.docker.com/ "https://cloud.docker.com") account.
2. Click on the user icon menu at the top right.
3. Select the organization that you would like to add a member too.
4. Select **Teams** in the bottom left menu bar.
5. Click the blue **add user** button.
6. Type in the Docker ID or email address for the user you would like to add.

The avatars on Docker Hub/Cloud are automatically linked to [Gravatar](https://en.gravatar.com "https://en.gravatar.com"). You can easily change your User or Organization avatar by following these steps:

1. Set up your Gravator here: [https://en.gravatar.com](https://en.gravatar.com "https://en.gravatar.com"). (skip this step if you have already set up your Gravatar)
2. Navigate to your Docker Hub/Cloud **[Account Settings](https://hub.docker.com/account/settings/ "https://hub.docker.com/account/settings/")** to update your Gravatar email for the Organization or User account.
3. Fill up the **Gravatar Email** field with the email that you set up in Step 1.
4. Click on **Save** and refresh the page.

The Organization or User account should now have an avatar displayed.

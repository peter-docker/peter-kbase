## Question

How do I SSH into a Docker Cloud node on AWS?

## Answer

Just follow [this article](https://docs.docker.com/docker-cloud/tutorials/ssh-into-a-node/ "https://docs.docker.com/docker-cloud/tutorials/ssh-into-a-node/") to be able to access your nodes through SSH. If you have chosen a custom security group, remember to have port 22 opened!

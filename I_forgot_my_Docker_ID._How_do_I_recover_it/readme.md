To recover your Docker ID username and/or password, go to the [Docker ID password reset page](https://hub.docker.com/reset-password/ "https://hub.docker.com/reset-password/"), enter the email address associated with your Docker ID, and click "Reset password". You will receive an email that contains both a link to reset your password as well as your Docker ID username. You will need to click on the password reset link and reset your password, but you also now know your Docker ID username.

## Question

How long does it take to deploy a Packet.net device?

## Answer

Docker Cloud deploys Ubuntu 14.04 LTS images on both types. Type 1 takes up to 5 ~ 10 min to be ready, while type 3 takes 8 ~ 13 min. Packet.net guys are working hard to reduce these deployment times.

A Docker ID can be claimed if it qualifies for the [User Squatting Policy 4.3.](https://www.docker.com/docker-terms-service "https://www.docker.com/docker-terms-service")

If you would like to claim an inactive Docker ID, open a [Customer Service Request](/Support "Account Support Form") and include the Docker ID you would like to claim.

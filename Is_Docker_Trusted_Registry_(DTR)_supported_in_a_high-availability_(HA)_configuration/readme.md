## Question

* Is Docker Trusted Registry (DTR) supported in a high-availability (HA) configuration?
* Can I load balance between instances of DTR to provide HA capabilities?

## Answer

Yes. The latest version of DTR allows for high-availability and load balancers. Refer to [Docker Reference Architecture: Docker Datacenter Best Practices and Design ](https://success.docker.com/Architecture/Docker_Reference_Architecture%3A_Docker_EE_Best_Practices_and_Design_Considerations "Docker Reference Architecture: Docker Datacenter Best Practices and Design Considerations")[Considerations](https://success.docker.com/Architecture/Docker_Reference_Architecture%3A_Docker_EE_Best_Practices_and_Design_Considerations "Docker Reference Architecture: Docker Datacenter Best Practices and Design Considerations") for details.

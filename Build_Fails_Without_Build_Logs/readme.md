# Question

My Docker Hub build logs are failing. Why are there no build logs are available?

# Answer

This can happen when there is an issue in the Docker backend build services before the actual build starts.

If you're having this problem, [create a new support ticket](http://success.docker.com/support).

> **Note:** Please include your Docker username/organization and the name of the repository.
